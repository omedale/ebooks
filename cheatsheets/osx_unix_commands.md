###An A-Z Index of the Apple OS X command line

To learn more about a command. From terminal simple type...

`man 'name_of_command'`

You will get a man page with full discriptions

Commands marked • are bash built-in commands. bash is the default shell, it runs under Darwin the open source core of OS X.
